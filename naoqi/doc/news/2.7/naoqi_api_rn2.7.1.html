NAOqi API & SDKs - 2.7.1¶

New: Qi SDK for Android¶

To discover the new way of programming for Pepper see the Pepper SDK for Android.

Core¶

ALConnectionManager: new methods¶

  • ALConnectionManagerProxy::setServiceIPv4
  • ALConnectionManagerProxy::setServiceDomains
  • ALConnectionManagerProxy::setServiceNameservers
  • ALConnectionManagerProxy::setServiceProxy
  • ALConnectionManagerProxy::provisionService
  • ALConnectionManagerProxy::provisionedServices
  • ALConnectionManagerProxy::removeProvisionedService

ALConnectionManager: deprecated method¶

  • ALConnectionManagerProxy::setServiceConfiguration

    Use ALConnectionManagerProxy::setServiceIPv4, ALConnectionManagerProxy::setServiceDomains ALConnectionManagerProxy::setServiceNameservers, or ALConnectionManagerProxy::setServiceProxy instead.

ALTabletService: disabled method¶

The tablet volume is now handled in Android settings screen. There is no API to change it. ALTabletService::setVolume does nothing anymore.

ALTabletService: deprecated module¶

This module is deprecated.

Use the Pepper SDK for Android instead.

Interaction engines¶

Collaborative Dialog uses the Semantic Engine¶

During a Collaborative Dialog, the Semantic Engine is activated. So, be aware of the additional content it brings.

For further details, see:

  • Semantic Engine & Collaborative Dialog,
  • Semantic Engine & QiChat rules,
  • Priority among rules in a Collaborative Dialog.

ALDialog: new events¶

  • Dialog/Scope()
  • Dialog/MainDialogStarted()

Emotion¶

ALMood: new property attentionZone¶

A new property ALMood::attentionZone allows the robot to know where the focused user is looking. The frame of reference is the user.

Motion¶

ALNavigation: new method¶

  • ALNavigationProxy::startMapping
