Detailed Description

template<typename N, typename A>
struct qi::IncrBoundedRangeAction< N, A >

Idem IncrBoundedRange but an Action performs the increment. Useful if you don't want to add an operator++ to N.

Use example (with sum taking a Range):

auto n = sum(incrRange(1, 64, [](int& i){i *= 2;})); // be careful to not step over the limit

Incrementable N, Action<N> A
