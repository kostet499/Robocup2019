Detailed Description

template<typename T, typename N>
struct qi::RepeatRange< T, N >

Repeat the same value n times.

Use example (with concat taking a Range) : auto s = concat(repeatRange(string("Y'a quelqu'un ? "), 3));

Regular T, Incrementable N
