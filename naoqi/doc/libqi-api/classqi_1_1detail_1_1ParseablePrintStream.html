Detailed Description

Provides facilities to print informations about types into a stream in parseable formatting. This class is movable but not copyable, thus not regular.

Example: ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~/

  • ParseablePrintStream stream(std::cout, DisplayHiddenMembers::Display);
  • MetaObject object = ...;
  • stream << object;
  • MetaMethod method = ...;
  • stream << method.name(); // no overload for MetaMethod, just print its name
  • ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~/
